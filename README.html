1 ESP8266

1.1 Overview

The ESP8266 is a low-cost micro-controller with built-in WiFi manufactured by Expressif Systems. This repository contains a few small Python programs I have wrote for this device.

1.2 Development Kit

I'm using the Heltec Wifi Kit 8 development board, which conveniently has a built-in 0.91 inch OLED screen.

1.3 Installation

In order to run these programs you must install the MicroPython firmware on your device, instructions can be found here. To upload the scripts I suggest using mpy-upload tool included in mpy-utils, but there are several methods. Remember to upload a main.py file, which calls the program (i.e. import myprogram). I've removed the information about my Wifi network and API tokens, so you will have to fill these in manually with your own.

1.4 Description

1.4.1 worldtime.py

This program simply displays the time in different parts of the world (Toronto, London, and China). It uses the network time protocol (NTP).

1.4.2 christmas.py

This is a count-down to Christmas! It also uses NTP.

1.4.3 airquality.py

This displays the real-time air quality and temperature for your current location (based on your IP address). It uses the aqicn.org API to get the information and updates every minute. You'll have to get your own API token to use it, but this is very easy and also free.
